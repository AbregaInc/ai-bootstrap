How it works

1

Copy the command

Click the Copy button above. This copies the install command to your clipboard.

2

Open Terminal

Terminal is an app on your Mac that lets you type commands. Here's how to open it:

  • Hold down the Command (‚åò) key and press Spacebar
  • A search box called "Spotlight" will appear
  • Type the word Terminal
  • Press Enter (or click on Terminal in the results)
  • A window with a blinking cursor will open ‚Äî that's Terminal!
3

Paste and run

Click inside the Terminal window, then:

  • Hold Command (‚åò) and press V to paste
  • You'll see the command appear in the Terminal
  • Press Enter to run it
4

Follow the prompts

The script will guide you through each step with clear explanations. It will ask your permission before installing anything. Just read what it says and type y (yes) or n (no) when asked.

Tip: When it asks for your password, type your Mac login password. You won't see any characters appear ‚Äî that's normal! Just type it and press Enter.

Requirements

  • macOS (Apple Silicon or Intel)
  • Admin password for your Mac
  • Internet connection
